about Label

2017-08-09

人类的天性就是喜欢把复杂的事物简单化，以此来简化对世界的认知，但是我们的思维常常会因此而下错误的判断。简单来说为什么会有地图炮，就是因为人们倾向于记住某一重大事件而对一个群体产生错误地判断，也就是心理学上说的易得性启发（availability heuristic），例如：河南人偷井盖事件、 莆田医院事件。这些重大的事件让人们印象深刻，也就很轻易地让人们对这个地域群体打上“人品不行、骗子”等标签。

还有一种标签就是对“人”打上标签，因为人性是复杂的。 很多女生选择相处或者相信一个人，通常会通过星座、血型、家庭背景、金钱等标签化的信息去定义一个人，这是简化对复杂人性认知的一种方式。其实也可以说是“懒得思考”，但其实细想这些东西都不太本质。一个人的价值观、对事业的追求与热爱、对待一件事的专注力与持久力，对复杂事件的解决能力和对事物的耐心程度，难道这些不才是更本质的能力吗。

但大多数女生的第一反应是，这些你让我怎么判断嘛…

所以说很多时候标签是难消除或者永远不会消失的东西，其实只要人们要花足够的耐心和时间去理解去相处，或者仅花一点时间去更客观的思考，标签就可以变得不存在。可是大多数时间往往就是“xx人不就是xxx”吗…（摊手）

自证预言和吸引力法则

2017-08-04

高中时候看过一本书叫《秘密》，讲的是吸引力法则，这本书当时在班里广为流传，读过的小伙伴纷纷觉得犹如神灵附身，想要什么就能有什么。闺蜜当时非常笃信这一原理，说自己上课忽然好想吃巧克力，就开始幻想自己吃巧克力的样子，到下课的时候还真有小伙伴掏出巧克力问她要不要吃…然后我就陷入深深地沉思，每天睡前都会幻想权志龙找上门来向我求婚…

其实吸引力法则说玄乎也有点玄乎，它的核心理论就是我们身处在这个宇宙之中，我们的所思所想每分每秒都在给宇宙发射一个信号，只要我们的信念够强大，宇宙就会对我们所发射的这个信号有所反馈。

让我开始对这一理论产生怀疑的当然就是大概我也坚持了一年左右每晚睡前念叨着权志龙，可是他却迟迟不肯找上来门这件事…

直到后来开始学心理学，里面有个叫“自证语言”这样的概念，一个例子就是：很多女生在上高中前，一些父母会说女孩子到高中成绩就会下滑了，到了高中就没有男生聪明了，女孩子还是选文科，理科肯定学不过男生的等等。这样的言论会很直接导致一个后果，女生在还没有接触这个前就觉得自己不行，以至于到后来越来越不行。相反，听到过这样言论的男生会越来越自信反而成绩越来越好。而这就是“自证语言”。

但实际上重点高中的女生理科和数学成绩超越男生是不要太正常的事情..想想生活在男权社会下的女生真的太不容易了，从小会遭遇到被猥亵的危险外，还要被认为“笨”和不如男生。

那么实际上“自证预言”和“吸引力法则”在某种程度上讲的就是一件事：心理暗示。

很多人会在缺乏自信或者紧张的时候进行一下轻微的心理暗示，却远远没想过其实“自证语言”可以在日常生活中帮助我们许多。
在开始学习一个新领域知识的时候不要先入为主打上“难”或者“学不会”这样的标签，看似小小的一个概念会对我们的大脑产生持续的影响，让后面的学习会越来越难，越来越学不会。如果打上“很容易” so easy 那自然…

所以这样看来，王健林所说的“先定个小目标，赚他一个亿。”也不是没有道理的嘛~

七月总结和思考

2017-08-02

1.七月出差了十天左右，认识了两个新的舞蹈老师，都是大学老师。聊了挺多，他们跟公司的领导差不多大，但是穿衣打扮和思维更年轻化（这是当然!）交流和沟通更容易，所以职业和日常跟谁呆在一起对人的影响真大。

2.这次出差最后悔的事情是忘记带本书出去，连kindle都忘了带。十天白白浪费了，决定不再出现这样的情况，就再也不出差。

3.七月就读完了一本书《本杰明·富兰克林传》，里面他自己写的完美人格计划，印象很深。我们从小就被教育接受自身的不完美，并且也知道没有完美人格的存在，更多时候也就有种放任自由的态度了。我就这样，爱咋咋地吧。大家都很浮躁..但是看了这本书就会感到好惭愧。一个人对自己的修养，和道德是如此严格要求，甚至都列了计划，列了表。还有他对生活的态度，对事物的热爱都很值得学习。 感觉是今年目前为止对我产生影响最大的一本书。

4.基本搞懂了区块链是什么，简单来说就是支撑比特币的底层算法，这种算法是无服务器，且去中心化的， 只要留下痕迹，就没办法消除。因此区块链技术可以运用在很多方面，Zeronet是一个，更期待后续的更多。

5.竟然开始对程序感兴趣，跟着教程一步步搭建了github，很好玩。彭彭买了本《“笨方法”学python》给我，果然很好玩。

6.使用“衣二三”这个app开始租衣服穿，发现一开始租衣服穿以后就对买衣服失去兴趣了，很好很节约很环保。 对比另一家“女神派”app，虽然女神派app里的品牌和衣服比“衣二三”多很多，也有针对不同用户进行价格分阶。但是整个app的设计很杂有点乱，用户体验很差，模特照片也不是统一拍的。就算都是我喜欢的牌子，也不会去使用。

这个社会的女生真的挺不容易的

2017-08-02

记得之前有次跟室友几个聊起来，讲到几个人多多少少都有遇到变态和遭遇过猥亵的经历。

初中有次上学，印象中那天天气特别好，难得起得特别早就高高兴兴背着书包上学去，我家离学校就五分钟的路程，平时路上都还会遇到同学结伴一起走或打个招呼什么的。 但是那天就是特别早，路上没几个人，对面就走来一个四十多岁的中年男子，我毫无防备也没有警觉，他忽然就伸出手狠狠抓了一把我的下体就从我身边走过去，我那时候就愣住了呆在原地，也不敢回头看他。但是眼神飘过，他好像还笑了一下。

这样的事情当然是谁也不敢说。

然后过了挺久，初中一个挺好的女生朋友课间生气地找我倾诉说自己早上遇到了一个变态，也是一下子摸她下体然后还对她笑。那时候就很佩服她的勇气，竟然有胆量可以说出来。

我还是谁也不敢说。

一直到了大学，就是前面写的跟室友聊的经历。原来一个寝室的六个人，有五个都遇到过。

有在HOLLISTER坐着等朋友买衣服，忽然出现个露阴癖掀开大衣的..

甚至都有小时候被亲戚乱摸过的..

这些都是小时候谁都不敢讲的事情，长大了后反而变得坦然。

虽然一直很讨厌女性被打上弱势群体的标签，但是生活在这个社会的女性和女生真的都太不容易了。大家都惊讶于原来自己并不孤单，原来你也遇到过这样的事情，好笑的背后是深深的悲哀。

想起很久之前在知乎上看过一个回答，说自己的小学在农村，都是留守儿童和老人的那种，小学老师禽兽不如对班里的女生进行猥亵，他们几个男生气不过可是又打不过那个男老师，他们太渺小了，最后只能是鼻青脸肿，然而那个禽兽还是继续猥亵。这个回答真正让人心痛的是后续，他说那几个小学经常遭到猥亵的女生长大后不是早早辍学去卖淫，就是有两个跳楼自杀了。

阳光照不到世界的另一面吧，在只有黑暗的世界摸爬滚打，伤痕累累，你却告诉我总有会阳光的，可是阳光，它什么时候才会出现。

六月读书总结

2017-07-25

六月就读了三本书：《学习之道》、《麦肯锡方法》和《影响力》。

《学习之道》

关于学习方法类的书籍看过很多，不外乎就是要建立概念跟概念之间的联结，及时复习以及拉长学习的周期。 这本书给我一个新的观点就是，当你解不出题的时候，你就别解，然后出去走走运动运动。我？？？？？？？ 这难道不是大家都知道的道理嘛， 你肯定在逗我。= = 这位号称以前是学渣后来变成大学霸的作者原来就是靠着这种神秘的方法，真是厉害厉害。

《影响力》

我发誓以后再也不看这些带点心理学基础知识的书了！！！包括一些学习方法类的书籍还有这本！！ 都是很基础的心理学知识翻来覆去的讲！！！我看一本戴维·迈尔斯的《心理学》就可以全涵盖了，写的还更通俗易懂，我干嘛还要花这么多冤枉钱去看这些不痛不痒的啊！！！！！浪！费！钱！！

《麦肯锡方法》

这本书买了很久，六月才开始看。没想到很惊喜的嘛，里面关于一些逻辑还有工作上的方法都是非常实用的。跟麦肯锡推崇的另一本晦涩难读的《金字塔原理》不同，这本非常浅显，而且有点有趣。到后面竟然还有出差宝典，可以说确实是非常实用了。大概以后书荒了，还会翻出来再读读看，谁知道呢~

零基础使用hexo在github上搭建blog

2017-07-19

主要分为三个部分：

1.注册github账号并建repository仓库

2.环境配置和安装hexo

3.部署hexo到github上

1.注册github账号

1.1进入https://github.com/ 注册

然后点建立仓库

注意一下repository name 前缀名一定要跟 github昵称一样，具体格式是：
yourname.github.io

然后点击 create repository 就可以创建了

2.环境配置和安装hexo

2.1 下载node.js安装文件：

https://nodejs.org/en/download/

2.2 下载git安装文件：

https://git-scm.com/downloads

2.3 确认本地已经安装好npm和git

在电脑搜索CMD，并打开输入：

  1. git –version
  2. npm –version

确认下版本已经安装成功

2.4 安装hexo

  1. 新安装一个文件夹 可以命名为blog

  2. 在电脑搜索CMD，并打开输入： cd blog
    接着安装hexo,输入：npm install -g hexo -cli
    安装成功后以后，输入：hexo，会看到下面这样

  3. 在CMD中继续输入： npm install hexo-cli -g
    可能会看到一两个WARN，不用担心，继续输入：
    npm install hexo –save
    确认下版本，是否安装成功，输入：
    hexo –version
    如果看到这样，就说明安装成功

3.部署hexo到github

3.1配置Git的个人信息

如果你之前已经配置好git个人信息，请跳过这一个 步骤，
设置Git的user name和email 输入：

  1. git config –global user.name “username”

  2. git config –global user.email “注册邮箱地址”

3.2生成密钥

  1. 在cmd中输入：ssh-keygen -t rsa -C “Github的注册邮箱地址”
    一路Enter过来就好，待秘钥生成完毕，会得到两个文件id_rsa和id_rsa.pub，用带格式的记事本打开id_rsa.pub，Ctrl + a复制里面的所有内容，然后进入：https://github.com/settings/ssh：

  2. 将复制的内容粘贴到Key的输入框，随便写好Title里面的内容，点击Add SSH key按钮即可。

3.3配置deployment

  1. 在blog文件里找到_config.yml用Atom打开，没有的话 下载地址：https://atom.io/
    找到deployment，按照如下修改：
    deploy:
    type: git
    repo: git@github.com:yourname/yourname.github.io.git
    branch: master
    （注意type,repo,branch前面空两个，冒号后面空一格）

3.4生成并部署

  1. 在cmd中输入：
    hexo generate
    生成
    hexo deploy
    部署到github上.

没有问题的话进入yourname.github.io 就部署成功了。
